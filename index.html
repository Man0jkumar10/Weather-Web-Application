About the Project

WeatherApp is a professional web application designed to provide accurate and up-to-date weather forecasts for any location in the world. Utilizing the powerful OpenWeatherMap API, users can search for weather information by city name. The application features a modern, responsive design and smooth animations for an enhanced user experience.
